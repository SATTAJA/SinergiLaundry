#SinergiLaundry

**SinergiLaundry** is a modern and responsive laundry service landing page designed to help customers discover laundry services, view transparent pricing, and book directly via WhatsApp.

##Features

* Modern and responsive landing page
* Clean hero section
* Laundry services showcase
* Transparent pricing section
* Direct booking via WhatsApp
* Mobile-friendly design
* Smooth scrolling navigation

##Tech Stack

* Next.js
* TypeScript
* Tailwind CSS

##Sections

* Hero
* Services
* Pricing
* Contact
* WhatsApp Booking

  
This project was created for learning and portfolio purposes.
